# 최종 적용본 (청첩장 4페이지 + 카카오페이/토스 버튼)

## 추가된 것
1) '청첩장' 섹션: 4페이지 미리보기(2x2)
   - 누르면 전체 화면 뷰어로 열림
   - 좌/우 버튼, 스와이프, 점(dot) 이동 지원

2) 계좌 섹션: 버튼 3개
   - 계좌 복사
   - 카카오페이 (계좌 복사 후 카카오페이 앱/웹 열기)
   - 토스 (계좌 복사 후 토스 앱/웹 열기)

## 주의
- 카카오페이/토스는 공식적으로 '계좌번호까지 자동 입력' 딥링크가 환경에 따라 제한될 수 있어
  버튼은 '계좌 복사 + 앱 열기' 방식으로 안정적으로 동작하도록 구성했습니다.
